📡 WiFi / ESP32 연결 설정 ESP32에 전달됨
저장 후 ESP32가 다음 부팅 시 GET /esp32/config로 자동 수신합니다.
ESP32와 Raspberry Pi는 같은 공유기에 연결되어 있어야 합니다.
ESP32 config.h 의 SERVER_PORT 와 일치시키세요
이 시간 이상 무응답 시 "연결 끊김" 표시
Raspberry Pi IP 확인 방법
hostname -I 명령으로 확인 후 ESP32 config.h 의 SERVER_IP 에 입력하세요.
